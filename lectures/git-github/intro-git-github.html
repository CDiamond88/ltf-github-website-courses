What are Git and Github

  • Git was created for collaborative software development. It tracks all changes made to files, and can merge together changes made separately by different people.
  • Github is a web platform owned by Microsoft that hosts Git projects and provides complementary tools.

Git steps in command form

  • git add <filename> : stage a file to be committed
  • git commit : commit your changes to the repository with a message
  • git push : push your changes to a remote repository like Github (backup)
  • git pull : pull any remote changes on Github to your computer.

Let's collaborate on this workshop

  1. Fork the repository.
  2. Enable Github pages
  3. On your version of the repository, edit workshop files (*.md) to improve the workshop on Github.
  4. Or, if you installed Git you can clone your version of the repository to your local computer and edit with your text editor of choice. Recommended if you plan on using these tools a lot.
